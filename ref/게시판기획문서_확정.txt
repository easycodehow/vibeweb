## 게시판 최종 기획서

---

### 프로젝트 맥락
AI바이브웹 사이트의 커뮤니티 섹션 — 자유게시판
- 기술스택: HTML5, CSS3, JavaScript + Supabase (DB/인증) + Vercel (배포)

---

### 기능 범위
- CRUD : 글 작성 / 조회 / 수정 / 삭제
- 댓글 기능
- 파일 첨부 기능
- 검색 기능
- 페이지네이션 : 페이지 번호 10개씩 표시, 다음 버튼으로 새 10개 세트

---

### 권한 시스템

| 구분           | 읽기 | 글쓰기 | 수정 | 삭제                |
|----------------|------|--------|------|---------------------|
| 비로그인       | 가능 | 불가   | 불가 | 불가                |
| 로그인 (본인 글) | 가능 | 가능   | 가능 | 가능                |
| 로그인 (타인 글) | 가능 | 가능   | 불가 | 불가                |
| 관리자         | 가능 | 가능   | 불가 | 모든 글 삭제 가능   |

- 로그인 방법: 회원가입 시 만든 아이디 + 비밀번호

---

### 목록 화면 표시 항목
글번호 / 제목 / 작성자(아이디) / 작성일시 / 조회수

---

### DB 테이블 구조

게시글 테이블
- id : 글번호 (PK)
- title : 제목
- content : 내용
- author_id : 글쓴사람 (회원 FK)
- created_at : 작성날짜
- views : 조회수

추가 필요 테이블: 회원(users) / 댓글(comments) / 첨부파일(attachments)
